# Instructions

`$ unzip hatch_demo.zip`  
`$ dosbox`  

Within the DOSBox environment:
```
Z:\> IMGMOUNT.COM c hatch.img
Z:\> C:
C:\> UDEMO.EXE
```
You will see a warning about being in the root directory; ignore that and press enter.
